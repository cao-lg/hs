# 2026年安徽省高考志愿参考系统 - 部署指南

```

```

## 快速部署到Cloudflare Pages

### 方式一：Wrangler CLI（推荐）

1. 安装 Wrangler：
```bash
npm install -g wrangler
```

2. 部署项目：
```bash
cd gaokao-zy
wrangler pages deploy .
```

首次部署会提示创建项目，按提示操作即可。

### 方式二：GitHub 集成

1. 将 `gaokao-zy` 文件夹推送到 GitHub 仓库
2. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
3. 进入 Workers & Pages → 创建应用程序 → 选择 Pages
4. 选择 "连接到 Git 仓库"
5. 选择你的 GitHub 仓库并配置构建命令（留空，直接部署静态文件）
6. 部署完成

### 方式三：直接上传

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 进入 Workers & Pages → 创建应用程序 → 选择 Pages
3. 选择 "直接上传"
4. 将整个 `gaokao-zy` 文件夹拖入上传区域
5. 部署完成

## 用户使用指南

### 输入成绩获取推荐

1. 打开网站首页
2. 选择科目类型（历史类/物理类）
3. 输入高考分数和省内排名
4. 点击"智能推荐"按钮
5. 系统将显示三类推荐院校：
   - **稳妥推荐**：录取希望较大
   - **冲一冲**：有一定把握
   - **冒险一试**：分数有一定差距

### 查看院校详情

1. 在推荐列表中点击"查看详情"
2. 查看院校基本信息、优势专业、历年分数线
3. 点击返回继续查看其他院校

### 专业查询

1. 进入"专业查询"页面
2. 可按院校筛选或搜索专业名称
3. 查看专业录取分数和位次
